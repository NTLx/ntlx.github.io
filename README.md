欢迎来到我的个人知识库！这是一个基于 [Docsify](https://docsify.js.org) 构建的静态文档网站，用于记录和分享我在系统管理、软件使用、生物信息学等领域的学习笔记和技术总结。

## 📚 内容概览

本项目主要包含以下几类技术文档：

- **System Administration (系统管理)**
  - Linux 发行版配置 (Arch Linux, CentOS, Manjaro, Ubuntu)
  - 服务器与集群管理 (Slurm, OpenMPI, UNRAID, VMWare)
  - 网络与代理服务 (Shadowsocks, Privoxy, Proxychains, ZeroTier)
  - 实用工具部署 (n8n, KMS)

- **Software Usage (软件使用)**
  - 命令行工具 (CLI, Bash, ZSH, Tmux)
  - 编辑器与开发工具 (VIM 配置与插件, Git, EditorConfig)

- **Bioinformatics (生物信息学)**
  - Snakemake 流程管理与性能分析

- **Services (服务)**
  - LLMs API Service

## 🚀 如何使用

### 在线浏览
你可以直接访问 GitHub Pages 进行浏览：
[https://ntlx.github.io/](https://ntlx.github.io/)

### 本地运行
如果你想在本地查看或调试文档，可以使用 `docsify-cli`：

1.  **安装 Docsify CLI** (需要 Node.js 环境):
    ```bash
    npm i docsify-cli -g
    ```

2.  **克隆仓库**:
    ```bash
    git clone https://github.com/NTLx/ntlx.github.io.git
    cd ntlx.github.io
    ```

3.  **启动本地服务器**:
    ```bash
    docsify serve docs
    ```

4.  **访问**:
    打开浏览器访问 `http://localhost:3000`。

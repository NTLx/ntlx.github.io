# AI 大模型代理服务使用指南

## 📖 简介

[**API Service**](https://api.ntlx.top/) 是一个专为开发者和用户提供的大模型 (LLM) API 代理服务平台。

- **核心价值**：旨在解决 OpenAI 等服务在特定地区的访问限制问题。通过本平台，您无需复杂的网络配置（如科学上网），即可直接调用 OpenAI 的 API 服务。
- **集成工具**：平台内置了一款功能强大的大模型聊天客户端，您可以点击顶部导航栏的 **“聊天”** 按钮，或直接访问 [Chat Client](https://ai.ntlx.top/) 进行体验。
- **服务范围**：目前主要提供 OpenAI API 的代理服务。如需特定版本的 OpenAI 模型或其他大模型支持，请联系管理员：`lx3325360@outlook.com`。

## 🚀 使用说明

### 1. 注册账号

您可以通过以下两种方式获取账号：

- **直接注册**：
    1. 访问 [API Service](https://api.ntlx.top/)。
    2. 点击右上角的 **“登录”** 按钮，然后选择 **“点击注册”**。
    3. 或者直接使用 **GitHub** 账号快捷登录。

- **邀请注册** (推荐)：
    - 使用邀请链接注册，邀请人和被邀请人均可获得额外的免费额度奖励。
    - [点击此处使用邀请链接注册](https://api.ntlx.top/register?aff=WpNe)

### 2. 获取额度 (充值)

平台采用预充值模式，支持支付宝扫码支付。

1.  **进入充值页面**：点击顶部导航栏的 **“充值”** 按钮。
2.  **获取兑换码**：
    - 点击 **“获取兑换码”** 按钮，扫描弹出的支付宝二维码进行付款。
    - **费率**：固定汇率 **1 USD : 8 CNY** (每次充值固定为 $5)。
    - 付款后，兑换码将发送至您的注册邮箱。
3.  **兑换额度**：将邮箱收到的兑换码填入输入框，点击 **“兑换”** 即可。

> [!IMPORTANT]
> **扫码转账时请务必备注您的用户名！**
> 如果未备注用户名，可能导致款项无法自动核销或追溯，由此造成的损失平台概不负责。

![充值界面](https://cdn.jsdelivr.net/gh/NTLx/Pic/PicGo/202403201001290.jpg)

### 3. 管理令牌 (Token)

令牌 (Token) 是您调用 API 服务的唯一凭证，相当于您的“密钥”。

1.  **进入管理页**：点击顶部导航栏的 **“令牌”** 按钮。
2.  **创建令牌**：点击 **“添加新的令牌”**，设置名称、过期时间及额度限制（可选）。
3.  **复制令牌**：创建完成后，点击 **“复制”** 按钮将令牌保存到剪贴板。

![令牌列表](https://cdn.jsdelivr.net/gh/NTLx/Pic/PicGo/202403200952674.png)
![添加令牌](https://cdn.jsdelivr.net/gh/NTLx/Pic/PicGo/202403200953147.png)

### 4. 配置与使用

获取令牌后，您可以在任何支持 OpenAI 接口的客户端中使用本服务。

#### 配置参数
- **API Key (密钥)**: 填入您刚才复制的 `sk-xxxxxx` 令牌。
- **API Base URL (接口地址)**: `https://api.ntlx.top` (部分客户端可能需要加上 `/v1`，即 `https://api.ntlx.top/v1`)。

#### 示例：配置内置聊天客户端
1.  打开 [Chat Client](https://ai.ntlx.top/) 或点击平台顶部的 **“聊天”**。
2.  在设置中找到 **“自定义接口”** (Custom Endpoint) 选项。
3.  填入上述 **接口地址** 和 **密钥**。

> [!TIP]
> 如果您是从令牌管理界面直接点击 **“聊天”** 跳转过来的，系统通常会提示自动填充配置，点击 **“确认”** 即可直接开始使用。
